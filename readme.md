### 개요

AirTraffic은 WebGL을 이용하여 현재 항공기의 위치와 날씨 정보를 시각화한 프로젝트입니다.

### 담당 기능

1. 팀원들이 작업하기 쉽도록 래퍼 라이브러리를 작성
2. 날씨 정보와 항공기 위치 정보 표현
3. 항공기 (빨간 점) 클릭 시 상세 정보 출력
4. 메르카토르 프로젝션을 사용하는 날씨 텍스처를 등장방형도법으로 변환
